# EXCHANGE

Esta es una aplicacion para IOS desarrollada en Objective C. Permite realizar cambios de moneda con base en el dolar en Libra esterlina, Euro, Yen y Real brasile√±o.

Hace uso de la libreria AFnetworking. Antes de correr el proyecto, es necesario agregarla de la siguiente manera.

```
$ sudo gem install cocoapods
$ pod setup
```

posterior a esto se instala la dependencia:

```
$ pod install
```

Para abrir el proyecto

```
open exchange.xcworkspace
```
